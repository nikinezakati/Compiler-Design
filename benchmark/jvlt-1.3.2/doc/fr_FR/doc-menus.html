Le menu Fichier

Fichier > Nouveau
Crée un nouveau dictionnaire. Comme jVLT ne peut travailler qu’avec un seul dictionnaire à la fois, vous devrez auparavant enregistrer le dictionnaire ouvert.

Fichier > Ouvrir
Affiche une fenêtre de dialogue qui vous permet d’ouvrir un dictionnaire.

Fichier > Ouvrir un fichier récent
Affiche les fichiers les plus récemment ouverts dans un sous-menu. Sélectionner un article de menu correspond à ouvrir le fichier correspondant.

Fichier > Enregistrer
Cette commande enregistre votre dictionnaire.

Fichier > Enregistrer sous…
Saves a dictionary letting you specify a custom Fichier  name and location.

Fichier > Imprimer, Fichier > Aperçu avant impression…
L’impression est décrite ici.

Fichier > Importer…
Ouvre une fenêtre de dialogue qui vous permet d’importer un dictionnaire donné, soit dans un format jvlt ou CSV (comma separated values).

Fichier > Exporter…
Ouvre une fenêtre de dialogue qui vous permet d’exporter des mots et exemples dans un fichier soit au format jvlt, soit au format CSV (comma separated values).

Fichier > Quitter
Quitte l’application. Si le dictionnaire actuellement ouvert a été modifié, il vous sera demandé si vous souhaitez en enregistrer les modifications.

Le menu Édition

Édition > Annuler
Annule la dernière commande ayant changé le contenu du dictionnaire. Ces commandes sont l’ajout, l’édition et la suppression de mots ou d’exemples, et l’application des résultats d’un quiz.

Édition > Rétablir
Rétablit la dernière étape d’annulation.

Édition > Propriétés
Édite les propriétés d’un dictionnaire. Il n’y a actuellement qu’une seule propriété, la langue. Choisir la langue n’est pas obligatoire, cela ne fait que déterminer quels champs personnalisés (genre des noms, conjugaison des verbes, etc) vous pouvez choisir dans la fenêtre de dialogue avancée pour les mots.

Le menu Outils

Outils > Réinitialiser les statistiques
Cet article de menu ouvre une fenêtre de dialogue qui vous permet d’effacer les statistiques (derniers mots interrogés, nombre d’erreurs, etc) pour tous les mots ou bien seulement pour les mots actuellement affichés dans la liste de mots (vous pouvez utiliser un filtre pour réduire le nombre de mots qui y sont affichés).

Outils > Afficher la console des erreurs
En sélectionnant cet article de menu, vous pouvez ouvrir une boîte de dialogue qui affiche les messages d’erreur que jVLT peut produire. En principe cette fenêtre devrait être vide, mais si vous constatez un comportement inattendu (si le fait de cliquer sur un bouton ne produit aucun effet, par exemple), vous pouvez consulter la console des erreurs et envoyer son contenu à l’auteur du logiciel.

Outils > Préférences…
Cette commande vous permet d’éditer certains paramètres.

Le menu Aide

Aide > Aide jVLT 
Cette command ouvre une nouvelle fenêtre affichant le manuel d’utilisation de jVLT.

Aide > À propos de jVLT
Affiche la version et des informations concernant l’application (auteur).
